# 02 — Principle

The full principle statement lives at [`docs/methodology/02-principle.md`](../../../docs/methodology/02-principle.md) in this repository.

In short: two principles govern the early-stage skills.

1. **Each stage answers only its own destination, leaving the next stage's path blank.** Foundation answers why/what/initial-tooling but not how to slice. The pattern repeats at every stage.
2. **The AI does not fill blanks by inference.** Within destination scope, only what the user explicitly stated is recorded. Missing values become `TBD: <reason>`, never guessed. This enforces Principle 1.
